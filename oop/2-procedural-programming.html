面向过程的Java编程

从Java基础语法到流程控制的完整学习指南

面向过程篇

在前面，我们了解了计算机的相关知识，也搭建好了Java开发环境，从本章开始，我们就要正式进入到Java语言的学习当中了。

首先，通过 file->新建->项目 创建一个名为MyHelloWorldProject的项目

在开始之前，我们需要先了解一下什么是面向过程的编程思想。

面向过程，顾名思义，就是面向某个过程去编程，比如我们要做一件事情，那么我们就需要考虑，完成这件事情需要哪些步骤，然后我们一步一步地去完成就行了。

比如我们要计算1+2+3+...+100的结果，那么面向过程的思想就是：

  1. 定义一个变量用于保存结果
  2. 从1开始，一直加到100
  3. 输出结果

我们只需要按照这个流程，一步一步地去实现就可以了。实际上我们之前编写的程序，都是按照面向过程的思想来编写的。

当然，除了面向过程之外，还有面向对象的编程思想，我们会在后面的章节中详细介绍，这里暂时不做讲解。

Java程序基础

程序代码基本结构

我们在前面已经编写了第一个Java程序，我们来回顾一下：

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

实际上这就是一个Java程序最基本的结构，我们来详细分析一下：

  • public class Main - 这是一个类的定义，类名为Main
  • public static void main(String[] args) - 这是程序的入口方法
  • System.out.println("Hello World!"); - 这是一条输出语句

每个Java程序都必须有一个类，而且类名必须与文件名相同。程序的执行从main方法开始，这是程序的入口点。

注释

在编写代码的过程中，我们可能需要对某些代码进行说明，这时我们就可以使用注释。注释是给程序员看的，不会被编译器执行。

Java中有三种注释方式：

单行注释

使用//开头，只能注释一行：

public class Main {
    public static void main(String[] args) {
        // 这是一个单行注释
        System.out.println("Hello World!");
    }
}

注意需要区分大小写，Java语言严格区分大小写，如果我们没有按照规则来编写，那么就会出现红色波浪线报错

只要源代码中存在报错的地方，就无法正常完成编译得到二进制文件，会提示构建失败

如果两段代码没有加上分号分割，那么编译器会认为这两段代码是同一句代码中的，即使出现换行或者是空格

这里我们尝试在中途换行和添加空格，因为没有添加分号，所以说编译器依然会认为是一行代码

如果添加了分号，即使在同一行，也会被认为是两句代码

如果直接写上文字的话，会导致编译不通过，因为这段文字也会被认为是程序的一部分

添加双斜杠之后（自动变成了灰色），后续的文本内容只要没有发生换行，那么都会被认为是一段注释

一旦发生换行那就不行了，需要重新添加注释符号

多行注释

使用/*开始，*/结束，可以注释多行：

public class Main {
    public static void main(String[] args) {
        /*
         * 这是一个多行注释
         * 可以写很多行
         */
        System.out.println("Hello World!");
    }
}

文档注释

使用/**开始，*/结束，主要用于生成API文档：

/**
 * 这是一个文档注释
 * @author 作者名
 * @version 1.0
 */
public class Main {
    /**
     * 程序入口方法
     * @param args 命令行参数
     */
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

变量与常量

变量的定义与使用

变量是用来存储数据的容器，在Java中，我们需要先定义变量，然后才能使用。

变量定义并赋值后的输出结果

常量的值不允许发生修改，编译时会出现错误

变量定义的语法格式：

数据类型 变量名 = 初始值;

例如：

public class Main {
    public static void main(String[] args) {
        int a = 10;        // 定义一个整数变量a，初始值为10
        double b = 3.14;   // 定义一个浮点数变量b，初始值为3.14
        String name = "张三"; // 定义一个字符串变量name，初始值为"张三"
        
        System.out.println(a);     // 输出变量a的值
        System.out.println(b);     // 输出变量b的值
        System.out.println(name);  // 输出变量name的值
    }
}

变量在使用前必须先定义，而且变量名必须符合标识符的命名规则：

  • 只能包含字母、数字、下划线和美元符号
  • 不能以数字开头
  • 不能使用Java关键字
  • 区分大小写

常量

常量是值不能改变的量，在Java中使用final关键字来定义常量：

public class Main {
    public static void main(String[] args) {
        final double PI = 3.14159;  // 定义一个常量PI
        final int MAX_SIZE = 100;   // 定义一个常量MAX_SIZE
        
        System.out.println(PI);
        System.out.println(MAX_SIZE);
        
        // PI = 3.14;  // 错误！常量不能重新赋值
    }
}

常量名通常使用全大写字母，多个单词之间用下划线分隔。

基本数据类型

Java是一种强类型语言，每个变量都必须声明其数据类型。Java中的数据类型分为两大类：

  • 基本数据类型：直接存储值
  • 引用数据类型：存储对象的引用

本节我们主要学习基本数据类型。

计算机二进制表示

在学习数据类型之前，我们需要先了解计算机是如何存储数据的。计算机内部使用二进制来表示所有数据。

二进制只有0和1两个数字，计算机中的每一位（bit）只能存储0或1。8个bit组成1个字节（byte）。

补码表示

对于有符号整数，计算机使用补码来表示负数：

  • 正数的补码就是其二进制表示
  • 负数的补码是其绝对值的二进制表示按位取反后加1

整数类型

Java提供了四种整数类型：

类型 大小 范围 默认值
byte 1字节 -128 到 127 0
short 2字节 -32,768 到 32,767 0
int 4字节 -2,147,483,648 到 2,147,483,647 0
long 8字节 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807 0L 
public class Main {
    public static void main(String[] args) {
        byte b = 100;
        short s = 1000;
        int i = 100000;
        long l = 100000000L;  // long类型需要在数字后面加L

        System.out.println("short: " + s);
        System.out.println("int: " + i);
        System.out.println("long: " + l);
    }
}

不同整数类型之间可以进行转换：

大的类型不能直接赋值给小的类型，会出现编译错误

当整数超出其类型能表示的最大值时，会发生溢出，正数变成负数

由于精度问题，浮点数计算可能不够精确

Java支持多种进制的整数常量：

浮点数的精度问题

由于浮点数的存储方式，可能会出现精度问题：

public class Main {
    public static void main(String[] args) {
        double d1 = 0.1;
        double d2 = 0.2;
        double sum = d1 + d2;
        
        System.out.println(sum);  // 可能不是0.3
        System.out.println(sum == 0.3);  // false
    }
}

字符编码

计算机中字符的编码经历了多个发展阶段：

  • ASCII：美国标准信息交换码，7位编码，128个字符
  • GB2312：中国国家标准，支持简体中文
  • GBK：扩展的GB2312，支持更多中文字符
  • Unicode：国际标准，支持世界上所有语言
  • UTF-8：Unicode的变长编码实现
  • UTF-16：Unicode的定长编码实现

Java内部使用UTF-16编码存储字符，每个char占用2个字节。

运算符

运算符是用来进行各种运算的符号。Java提供了丰富的运算符，包括算术运算符、关系运算符、逻辑运算符等。

赋值运算符

赋值运算符用于给变量赋值：

public class Main {
    public static void main(String[] args) {
        int a = 10;     // 基本赋值
        int b = a;      // 将a的值赋给b
        
        a += 5;         // 等价于 a = a + 5
        b -= 3;         // 等价于 b = b - 3
        
        System.out.println("a = " + a);
        System.out.println("b = " + b);
    }
}

算术运算符

算术运算符用于进行数学运算：

运算时会发生隐式类型转换，结果类型为较大的数据类型

运算符 描述 示例
+ 加法 a + b
- 减法 a - b
* 乘法 a * b
/ 除法 a / b
% 取模（求余） a % b 
public class Main {
    public static void main(String[] args) {
        int a = 10, b = 3;
        
        System.out.println("a + b = " + (a + b));  // 13
        System.out.println("a - b = " + (a - b));  // 7
        System.out.println("a * b = " + (a * b));  // 30
        System.out.println("a / b = " + (a / b));  // 3（整数除法）
        System.out.println("a % b = " + (a % b));  // 1（余数）
    }
}

整数除法只保留整数部分，小数部分被舍弃

当除法运算中有浮点数时，结果为浮点数

取模运算返回除法的余数

字符串拼接

当+运算符用于字符串时，表示字符串拼接：

public class Main {
    public static void main(String[] args) {
        String str1 = "Hello";
        String str2 = "World";
        String result = str1 + " " + str2;
        
        System.out.println(result);  // Hello World
        
        // 数字与字符串拼接
        int num = 100;
        String msg = "数字是：" + num;
        System.out.println(msg);  // 数字是：100
    }
}

字符串与数字拼接时，数字会自动转换为字符串

运算符优先级

当表达式中有多个运算符时，需要考虑优先级：

public class Main {
    public static void main(String[] args) {
        int result1 = 2 + 3 * 4;        // 14，不是20
        int result2 = (2 + 3) * 4;      // 20，括号改变优先级
        
        System.out.println(result1);
        System.out.println(result2);
    }
}

类型转换

在运算过程中，可能需要进行类型转换：

public class Main {
    public static void main(String[] args) {
        // 自动类型转换（隐式）
        int i = 100;
        double d = i;  // int自动转换为double
        
        // 强制类型转换（显式）
        double d2 = 3.14;
        int i2 = (int) d2;  // double强制转换为int，会丢失小数部分
        
        System.out.println("d = " + d);
        System.out.println("i2 = " + i2);
    }
}

大范围类型转换为小范围类型时需要强制类型转换

强制类型转换可能导致数据溢出或精度丢失

自增自减运算符

自增（++）和自减（--）运算符用于变量的递增和递减：

public class Main {
    public static void main(String[] args) {
        int a = 5;
        
        System.out.println("a = " + a);        // 5
        System.out.println("++a = " + (++a));  // 6，先自增再使用
        System.out.println("a++ = " + (a++));  // 6，先使用再自增
        System.out.println("a = " + a);        // 7
        
        System.out.println("--a = " + (--a));  // 6，先自减再使用
        System.out.println("a-- = " + (a--));  // 6，先使用再自减
        System.out.println("a = " + a);        // 5
    }
}

前置和后置自增自减的区别在于运算和使用的顺序

位运算符

位运算符直接对二进制位进行操作：

运算符 描述 示例
& 按位与 a & b
| 按位或 a | b
^ 按位异或 a ^ b
~ 按位取反 ~a
<< 左移 a << 2
>> 右移 a >> 2
>>> 无符号右移 a >>> 2 
public class Main {
    public static void main(String[] args) {
        int a = 12;  // 二进制：1100
        int b = 10;  // 二进制：1010
        
        System.out.println("a & b = " + (a & b));  // 8（1000）
        System.out.println("a | b = " + (a | b));  // 14（1110）
        System.out.println("a ^ b = " + (a ^ b));  // 6（0110）
        System.out.println("~a = " + (~a));        // -13
        
        System.out.println("a << 1 = " + (a << 1));  // 24（左移1位相当于乘以2）
        System.out.println("a >> 1 = " + (a >> 1));  // 6（右移1位相当于除以2）
    }
}

关系运算符

关系运算符用于比较两个值，返回布尔类型结果：

运算符 描述 示例
== 等于 a == b
!= 不等于 a != b
> 大于 a > b
< 小于 a < b
>= 大于等于 a >= b
<= 小于等于 a <= b 
public class Main {
    public static void main(String[] args) {
        int a = 10, b = 20;
        
        System.out.println("a == b: " + (a == b));  // false
        System.out.println("a != b: " + (a != b));  // true
        System.out.println("a > b: " + (a > b));    // false
        System.out.println("a < b: " + (a < b));    // true
        System.out.println("a >= b: " + (a >= b));  // false
        System.out.println("a <= b: " + (a <= b));  // true
    }
}

逻辑运算符

逻辑运算符用于布尔值的逻辑运算：

运算符 描述 示例
&& 逻辑与（短路） a && b
|| 逻辑或（短路） a || b
! 逻辑非 !a 
public class Main {
    public static void main(String[] args) {
        boolean a = true, b = false;
        
        System.out.println("a && b: " + (a && b));  // false
        System.out.println("a || b: " + (a || b));  // true
        System.out.println("!a: " + (!a));          // false
        System.out.println("!b: " + (!b));          // true
        
        // 短路特性
        int x = 5;
        boolean result = (x > 10) && (++x > 0);  // ++x不会执行
        System.out.println("x = " + x);  // x仍然是5
    }
}

三元运算符

三元运算符是一种简化的条件表达式：

条件 ? 值1 : 值2

如果条件为真，返回值1；否则返回值2。

public class Main {
    public static void main(String[] args) {
        int a = 10, b = 20;
        
        // 使用三元运算符找出较大值
        int max = (a > b) ? a : b;
        System.out.println("较大值: " + max);
        
        // 等价的if-else语句
        int max2;
        if (a > b) {
            max2 = a;
        } else {
            max2 = b;
        }
        System.out.println("较大值: " + max2);
    }
}

运算符优先级总结

运算符的优先级从高到低排列：

  1. 后缀：() [] . ++ --
  2. 一元：++ -- ! ~ + -
  3. 乘除：* / %
  4. 加减：+ -
  5. 位移：<< >> >>>
  6. 关系：< <= > >= instanceof
  7. 相等：== !=
  8. 按位与：&
  9. 按位异或：^
  10. 按位或：|
  11. 逻辑与：&&
  12. 逻辑或：||
  13. 三元：? :
  14. 赋值：= += -= *= /= %= &= ^= |= <<= >>= >>>=

流程控制

程序的执行流程默认是从上到下顺序执行的，但在实际编程中，我们经常需要根据不同的条件执行不同的代码，或者重复执行某些代码。这就需要用到流程控制语句。

代码块与作用域

在学习流程控制之前，我们需要先了解代码块和作用域的概念。

代码块是用花括号{}包围的一段代码：

public class Main {
    public static void main(String[] args) {
        {  // 这是一个代码块
            int x = 10;
            System.out.println(x);
        }  // 代码块结束
        
        // System.out.println(x);  // 错误！x在这里不可见
    }
}

变量的作用域是指变量可以被访问的范围。在Java中，变量的作用域由其声明的位置决定。

选择结构

if语句

if语句用于根据条件执行不同的代码：

if (条件) {
    // 条件为真时执行的代码
}
public class Main {
    public static void main(String[] args) {
        int score = 85;
        
        if (score >= 90) {
            System.out.println("优秀");
        }
        
        if (score >= 60) {
            System.out.println("及格");
        }
    }
}

if-else语句

if-else语句提供了两个分支：

if-else语句的执行流程图

public class Main {
    public static void main(String[] args) {
        int score = 55;

switch语句

switch语句适用于多分支选择，特别是当需要根据一个变量的不同值执行不同代码时：

switch (目标) {
    case 匹配值1:
        代码...
        break;
    case 匹配值2:
        代码...
        break;
    default:
        默认代码...
}
public class Main {
    public static void main(String[] args) {
        char grade = 'B';
        
        switch (grade) {
            case 'A':
                System.out.println("去尖子班！准备冲刺985大学！");
                break;
            case 'B':
                System.out.println("去平行班！准备冲刺一本！");
                break;
            case 'C':
                System.out.println("去职高深造。");
                break;
            default:
                System.out.println("去读职高，分流");
        }
    }
}

switch语句根据等级给出不同的建议

注意：每个case后面都要加break，否则会继续执行下一个case的代码。

for (初始化; 条件; 更新) {

for循环的执行过程：

  1. 执行初始化语句（只执行一次）

  5. 回到第2步

while循环

while循环适用于不确定循环次数，只知道循环条件的情况：

while (条件) {
    循环体;
}
public class Main {
    public static void main(String[] args) {
        int i = 100;
        
        while (i > 0) {
            System.out.println(i);
            i /= 2;  // 每次循环都除以2
        }
    }
}

while循环不断将数值除以2直到为0

do-while循环

do-while循环先执行循环体，再判断条件：

do {
    循环体;
} while (条件);
public class Main {
    public static void main(String[] args) {
        int i = 0;
        
        do {
            System.out.println("Hello World!");
            i++;
        } while (i < 3);
    }
}

do-while循环至少执行一次循环体

循环控制语句

break语句

break用于提前终止循环：

public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i == 5) {
                break;  // 当i等于5时终止循环
            }
            System.out.println(i);
        }
        System.out.println("循环结束");
    }
}

break语句在i等于5时终止循环

continue语句

continue用于跳过当前循环的剩余部分，直接进入下一次循环：

public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) {
                continue;  // 跳过偶数
            }
            System.out.println(i);  // 只打印奇数
        }
    }
}

continue语句跳过偶数，只打印奇数

标签break和continue

在嵌套循环中，可以使用标签来指定break或continue的目标：

public class Main {
    public static void main(String[] args) {
        outer: for (int i = 1; i <= 3; i++) {
            inner: for (int j = 1; j <= 3; j++) {
                if (i == 2 && j == 2) {
                    break outer;  // 跳出外层循环
                }
                System.out.println("i=" + i + ", j=" + j);
            }
        }
        System.out.println("循环结束");
    }
}
