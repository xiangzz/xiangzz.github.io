运算符

运算符是用来进行各种运算的符号。Java提供了丰富的运算符，包括算术运算符、关系运算符、逻辑运算符等。

赋值运算符

赋值运算符用于给变量赋值：

public class Main {
    public static void main(String[] args) {
        int a = 10;     // 基本赋值
        int b = a;      // 将a的值赋给b
        
        a += 5;         // 等价于 a = a + 5
        b -= 3;         // 等价于 b = b - 3
        
        System.out.println("a = " + a);
        System.out.println("b = " + b);
    }
}

算术运算符

算术运算符用于进行数学运算：

运算时会发生隐式类型转换，结果类型为较大的数据类型

运算符 描述 示例
+ 加法 a + b
- 减法 a - b
* 乘法 a * b
/ 除法 a / b
% 取模（求余） a % b 
public class Main {
    public static void main(String[] args) {
        int a = 10, b = 3;
        
        System.out.println("a + b = " + (a + b));  // 13
        System.out.println("a - b = " + (a - b));  // 7
        System.out.println("a * b = " + (a * b));  // 30
        System.out.println("a / b = " + (a / b));  // 3（整数除法）
        System.out.println("a % b = " + (a % b));  // 1（余数）
    }
}

整数除法只保留整数部分，小数部分被舍弃

当除法运算中有浮点数时，结果为浮点数

取模运算返回除法的余数

字符串拼接

当+运算符用于字符串时，表示字符串拼接：

public class Main {
    public static void main(String[] args) {
        String str1 = "Hello";
        String str2 = "World";
        String result = str1 + " " + str2;
        
        System.out.println(result);  // Hello World
        
        // 数字与字符串拼接
        int num = 100;
        String msg = "数字是：" + num;
        System.out.println(msg);  // 数字是：100
    }
}

字符串与数字拼接时，数字会自动转换为字符串

运算符优先级

当表达式中有多个运算符时，需要考虑优先级：

public class Main {
    public static void main(String[] args) {
        int result1 = 2 + 3 * 4;        // 14，不是20
        int result2 = (2 + 3) * 4;      // 20，括号改变优先级
        
        System.out.println(result1);
        System.out.println(result2);
    }
}

类型转换

在运算过程中，可能需要进行类型转换：

public class Main {
    public static void main(String[] args) {
        // 自动类型转换（隐式）
        int i = 100;
        double d = i;  // int自动转换为double
        
        // 强制类型转换（显式）
        double d2 = 3.14;
        int i2 = (int) d2;  // double强制转换为int，会丢失小数部分
        
        System.out.println("d = " + d);
        System.out.println("i2 = " + i2);
    }
}

大范围类型转换为小范围类型时需要强制类型转换

强制类型转换可能导致数据溢出或精度丢失

自增自减运算符

自增（++）和自减（--）运算符用于变量的递增和递减：

public class Main {
    public static void main(String[] args) {
        int a = 5;
        
        System.out.println("a = " + a);        // 5
        System.out.println("++a = " + (++a));  // 6，先自增再使用
        System.out.println("a++ = " + (a++));  // 6，先使用再自增
        System.out.println("a = " + a);        // 7
        
        System.out.println("--a = " + (--a));  // 6，先自减再使用
        System.out.println("a-- = " + (a--));  // 6，先使用再自减
        System.out.println("a = " + a);        // 5
    }
}

前置和后置自增自减的区别在于运算和使用的顺序

位运算符

位运算符直接对二进制位进行操作：

运算符 描述 示例
& 按位与 a & b
| 按位或 a | b
^ 按位异或 a ^ b
~ 按位取反 ~a
<<< /td> 左移 a << 2
>> 右移 a >> 2
>>> 无符号右移 a >>> 2 
public class Main {
    public static void main(String[] args) {
        int a = 12;  // 二进制：1100
        int b = 10;  // 二进制：1010
        
        System.out.println("a & b = " + (a & b));  // 8（1000）
        System.out.println("a | b = " + (a | b));  // 14（1110）
        System.out.println("a ^ b = " + (a ^ b));  // 6（0110）
        System.out.println("~a = " + (~a));        // -13
        
        System.out.println("a << 1 = " + (a << 1));  // 24（左移1位相当于乘以2）
        System.out.println("a >> 1 = " + (a >> 1));  // 6（右移1位相当于除以2）
    }
}

关系运算符

关系运算符用于比较两个值，返回布尔类型结果：

运算符 描述 示例
== 等于 a == b
!= 不等于 a != b
> 大于 a > b
<< /td> 小于 a < b
>= 大于等于 a >= b
<=< /td> 小于等于 a <= b 
public class Main {
    public static void main(String[] args) {
        int a = 10, b = 20;
        
        System.out.println("a == b: " + (a == b));  // false
        System.out.println("a != b: " + (a != b));  // true
        System.out.println("a > b: " + (a > b));    // false
        System.out.println("a < b: " + (a < b));    // true
        System.out.println("a >= b: " + (a >= b));  // false
        System.out.println("a <= b: " + (a <= b));  // true
    }
}

逻辑运算符

逻辑运算符用于布尔值的逻辑运算：

运算符 描述 示例
&& 逻辑与（短路） a && b
|| 逻辑或（短路） a || b
! 逻辑非 !a 
public class Main {
    public static void main(String[] args) {
        boolean a = true, b = false;
        
        System.out.println("a && b: " + (a && b));  // false
        System.out.println("a || b: " + (a || b));  // true
        System.out.println("!a: " + (!a));          // false
        System.out.println("!b: " + (!b));          // true
        
        // 短路特性
        int x = 5;
        boolean result = (x > 10) && (++x > 0);  // ++x不会执行
        System.out.println("x = " + x);  // x仍然是5
    }
}

三元运算符

三元运算符是一种简化的条件表达式：

条件 ? 值1 : 值2

如果条件为真，返回值1；否则返回值2。

public class Main {
    public static void main(String[] args) {
        int a = 10, b = 20;
        
        // 使用三元运算符找出较大值
        int max = (a > b) ? a : b;
        System.out.println("较大值: " + max);
        
        // 等价的if-else语句
        int max2;
        if (a > b) {
            max2 = a;
        } else {
            max2 = b;
        }
        System.out.println("较大值: " + max2);
    }
}

运算符优先级总结

运算符的优先级从高到低排列：

  1. 后缀：() [] . ++ --
  2. 一元：++ -- ! ~ + -
  3. 乘除：* / %
  4. 加减：+ -
  5. 位移：<<>> >>>
  6. 关系：< <=> >= instanceof
  7. 相等：== !=
  8. 按位与：&
  9. 按位异或：^
  10. 按位或：|
  11. 逻辑与：&&
  12. 逻辑或：||
  13. 三元：? :
  14. 赋值：= += -= *= /= %= &= ^= |= <<=>>= >>>=
