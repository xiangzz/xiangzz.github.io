Java程序基础

程序代码基本结构

我们在前面已经编写了第一个Java程序，我们来回顾一下：

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

实际上这就是一个Java程序最基本的结构，我们来详细分析一下：

  • public class Main - 这是一个类的定义，类名为Main
  • public static void main(String[] args) - 这是程序的入口方法
  • System.out.println("Hello World!"); - 这是一条输出语句

每个Java程序都必须有一个类，而且类名必须与文件名相同。程序的执行从main方法开始，这是程序的入口点。

注释

在编写代码的过程中，我们可能需要对某些代码进行说明，这时我们就可以使用注释。注释是给程序员看的，不会被编译器执行。

Java中有三种注释方式：

单行注释

使用//开头，只能注释一行：

public class Main {
    public static void main(String[] args) {
        // 这是一个单行注释
        System.out.println("Hello World!");
    }
}

注意需要区分大小写，Java语言严格区分大小写，如果我们没有按照规则来编写，那么就会出现红色波浪线报错

只要源代码中存在报错的地方，就无法正常完成编译得到二进制文件，会提示构建失败

如果两段代码没有加上分号分割，那么编译器会认为这两段代码是同一句代码中的，即使出现换行或者是空格

这里我们尝试在中途换行和添加空格，因为没有添加分号，所以说编译器依然会认为是一行代码

如果添加了分号，即使在同一行，也会被认为是两句代码

如果直接写上文字的话，会导致编译不通过，因为这段文字也会被认为是程序的一部分

添加双斜杠之后（自动变成了灰色），后续的文本内容只要没有发生换行，那么都会被认为是一段注释

一旦发生换行那就不行了，需要重新添加注释符号

多行注释

使用/*开始，*/结束，可以注释多行：

public class Main {
    public static void main(String[] args) {
        /*
         * 这是一个多行注释
         * 可以写很多行
         */
        System.out.println("Hello World!");
    }
}

文档注释

使用/**开始，*/结束，主要用于生成API文档：

/**
 * 这是一个文档注释
 * @author 作者名
 * @version 1.0
 */
public class Main {
    /**
     * 程序入口方法
     * @param args 命令行参数
     */
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

变量与常量

变量的定义与使用

变量是用来存储数据的容器，在Java中，我们需要先定义变量，然后才能使用。

变量定义并赋值后的输出结果

常量的值不允许发生修改，编译时会出现错误

变量定义的语法格式：

数据类型 变量名 = 初始值;

例如：

public class Main {
    public static void main(String[] args) {
        int a = 10;        // 定义一个整数变量a，初始值为10
        double b = 3.14;   // 定义一个浮点数变量b，初始值为3.14
        String name = "张三"; // 定义一个字符串变量name，初始值为"张三"
        
        System.out.println(a);     // 输出变量a的值
        System.out.println(b);     // 输出变量b的值
        System.out.println(name);  // 输出变量name的值
    }
}

变量在使用前必须先定义，而且变量名必须符合标识符的命名规则：

  • 只能包含字母、数字、下划线和美元符号
  • 不能以数字开头
  • 不能使用Java关键字
  • 区分大小写

常量

常量是值不能改变的量，在Java中使用final关键字来定义常量：

public class Main {
    public static void main(String[] args) {
        final double PI = 3.14159;  // 定义一个常量PI
        final int MAX_SIZE = 100;   // 定义一个常量MAX_SIZE
        
        System.out.println(PI);
        System.out.println(MAX_SIZE);
        
        // PI = 3.14;  // 错误！常量不能重新赋值
    }
}

常量名通常使用全大写字母，多个单词之间用下划线分隔。
