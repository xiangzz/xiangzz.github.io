流程控制

程序的执行流程默认是从上到下顺序执行的，但在实际编程中，我们经常需要根据不同的条件执行不同的代码，或者重复执行某些代码。这就需要用到流程控制语句。

代码块与作用域

在学习流程控制之前，我们需要先了解代码块和作用域的概念。

代码块是用花括号{}包围的一段代码：

public class Main {
    public static void main(String[] args) {
        {  // 这是一个代码块
            int x = 10;
            System.out.println(x);
        }  // 代码块结束
        
        // System.out.println(x);  // 错误！x在这里不可见
    }
}

变量的作用域是指变量可以被访问的范围。在Java中，变量的作用域由其声明的位置决定。

选择结构

if语句

if语句用于根据条件执行不同的代码：

if (条件) {
    // 条件为真时执行的代码
}
public class Main {
    public static void main(String[] args) {
        int score = 85;
        
        if (score >= 90) {
            System.out.println("优秀");
        }
        
        if (score >= 60) {
            System.out.println("及格");
        }
    }
}

if-else语句

if-else语句提供了两个分支：

if-else语句的执行流程图

public class Main {
    public static void main(String[] args) {
        int score = 55;
        
        if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }
    }
}

根据分数判断是否及格的执行结果

if-else if-else语句

多重if语句用于处理多个条件：

public class Main {
    public static void main(String[] args) {
        int score = 85;
        
        if (score >= 90) {
            System.out.println("优秀");
        } else if (score >= 80) {
            System.out.println("良好");
        } else if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }
    }
}

多重if-else语句的执行流程，按顺序判断条件

嵌套if语句

if语句可以嵌套使用：

public class Main {
    public static void main(String[] args) {
        int score = 25;
        
        if (score < 60) {  // 先判断不及格
            if (score > 30) {  // 在内层再嵌套一个if语句
                System.out.println("学习C++");
            } else {
                System.out.println("学习Java");
            }
        }
    }
}

嵌套if语句根据分数范围给出不同建议

switch语句

switch语句适用于多分支选择，特别是当需要根据一个变量的不同值执行不同代码时：

switch (目标) {
    case 匹配值1:
        代码...
        break;
    case 匹配值2:
        代码...
        break;
    default:
        默认代码...
}
public class Main {
    public static void main(String[] args) {
        char grade = 'B';
        
        switch (grade) {
            case 'A':
                System.out.println("去尖子班！准备冲刺985大学！");
                break;
            case 'B':
                System.out.println("去平行班！准备冲刺一本！");
                break;
            case 'C':
                System.out.println("去职高深造。");
                break;
            default:
                System.out.println("去读职高，分流");
        }
    }
}

switch语句根据等级给出不同的建议

注意：每个case后面都要加break，否则会继续执行下一个case的代码。

循环结构

循环语句用于重复执行某段代码，直到满足退出条件。

for循环

for循环适用于已知循环次数的情况：

for (初始化; 条件; 更新) {
    循环体;
}
public class Main {
    public static void main(String[] args) {
        // 打印1到10
        for (int i = 1; i <= 10; i++) {
            System.out.println("第" + i + "次循环");
        }
    }
}

for循环打印1到10的执行结果

随着循环的进行，i的值也会逐渐自增：

当i增长到2时，此时来到最后一轮循环，再继续向下运行，就不再满足循环条件了，所以说此时就会结束循环。

for循环的执行过程：

  1. 执行初始化语句（只执行一次）
  2. 判断条件是否为真
  3. 如果为真，执行循环体
  4. 执行更新语句
  5. 回到第2步

嵌套for循环

for循环可以嵌套使用：

public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 3; i++) {      // 外层循环
            for (int j = 1; j <= 3; j++) {  // 内层循环
                System.out.println("i=" + i + ", j=" + j);
            }
        }
    }
}

嵌套for循环的执行结果，外层循环控制行，内层循环控制列

while循环

while循环适用于不确定循环次数，只知道循环条件的情况：

while (条件) {
    循环体;
}
public class Main {
    public static void main(String[] args) {
        int i = 100;
        
        while (i > 0) {
            System.out.println(i);
            i /= 2;  // 每次循环都除以2
        }
    }
}

while循环不断将数值除以2直到为0

do-while循环

do-while循环先执行循环体，再判断条件：

do {
    循环体;
} while (条件);
public class Main {
    public static void main(String[] args) {
        int i = 0;
        
        do {
            System.out.println("Hello World!");
            i++;
        } while (i < 3);
    }
}

do-while循环至少执行一次循环体

循环控制语句

break语句

break用于提前终止循环：

public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i == 5) {
                break;  // 当i等于5时终止循环
            }
            System.out.println(i);
        }
        System.out.println("循环结束");
    }
}

break语句在i等于5时终止循环

continue语句

continue用于跳过当前循环的剩余部分，直接进入下一次循环：

public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) {
                continue;  // 跳过偶数
            }
            System.out.println(i);  // 只打印奇数
        }
    }
}

continue语句跳过偶数，只打印奇数

标签break和continue

在嵌套循环中，可以使用标签来指定break或continue的目标：

public class Main {
    public static void main(String[] args) {
        outer: for (int i = 1; i <= 3; i++) {
            inner: for (int j = 1; j <= 3; j++) {
                if (i == 2 && j == 2) {
                    break outer;  // 跳出外层循环
                }
                System.out.println("i=" + i + ", j=" + j);
            }
        }
        System.out.println("循环结束");
    }
}
