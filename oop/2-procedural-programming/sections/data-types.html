基本数据类型

Java是一种强类型语言，每个变量都必须声明其数据类型。Java中的数据类型分为两大类：

  • 基本数据类型：直接存储值
  • 引用数据类型：存储对象的引用

本节我们主要学习基本数据类型。

计算机二进制表示

在学习数据类型之前，我们需要先了解计算机是如何存储数据的。计算机内部使用二进制来表示所有数据。

二进制只有0和1两个数字，计算机中的每一位（bit）只能存储0或1。8个bit组成1个字节（byte）。

补码表示

对于有符号整数，计算机使用补码来表示负数：

  • 正数的补码就是其二进制表示
  • 负数的补码是其绝对值的二进制表示按位取反后加1

整数类型

Java提供了四种整数类型：

类型 大小 范围 默认值
byte 1字节 -128 到 127 0
short 2字节 -32,768 到 32,767 0
int 4字节 -2,147,483,648 到 2,147,483,647 0
long 8字节 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807 0L 
public class Main {
    public static void main(String[] args) {
        byte b = 100;
        short s = 1000;
        int i = 100000;
        long l = 100000000L;  // long类型需要在数字后面加L
        
        System.out.println("byte: " + b);
        System.out.println("short: " + s);
        System.out.println("int: " + i);
        System.out.println("long: " + l);
    }
}

类型转换

不同整数类型之间可以进行转换：

大的类型不能直接赋值给小的类型，会出现编译错误

直接编写的整数常量默认为int类型，超出int范围需要添加L后缀

当整数超出其类型能表示的最大值时，会发生溢出，正数变成负数

浮点类型

Java提供了两种浮点类型用于表示小数：

类型 大小 精度 默认值
float 4字节 约7位小数 0.0f
double 8字节 约15位小数 0.0

浮点数在计算机中采用IEEE 754标准存储：

IEEE 754标准定义了浮点数的存储格式，包括符号位、指数位和尾数位

小数常量默认为double类型，直接赋值给float会出错，需要添加F后缀

由于精度问题，浮点数计算可能不够精确

字符类型

字符类型用于表示单个字符：

每个字符都对应一个ASCII码值，比如字母A对应65

可以通过ASCII码值来输出对应的字符

public class Main {
    public static void main(String[] args) {
        char c1 = 'A';     // 直接使用字符常量
        char c2 = 65;      // 使用ASCII码值
        char c3 = '中';    // 中文字符
        
        System.out.println(c1);
        System.out.println(c2);
        System.out.println(c3);
    }
}

Java支持Unicode编码，可以直接使用中文字符

Java甚至支持使用中文作为变量名

字符类型只能表示单个字符，多个字符需要使用字符串

布尔类型

布尔类型只有两个值：true和false

布尔类型只能赋值true或false，其他值会编译错误

public class Main {
    public static void main(String[] args) {
        boolean flag1 = true;
        boolean flag2 = false;
        
        System.out.println(flag1);
        System.out.println(flag2);
    }
}

整数常量的表示

Java支持多种进制的整数常量：

public class Main {
    public static void main(String[] args) {
        int decimal = 100;      // 十进制
        int octal = 0144;       // 八进制，以0开头
        int hex = 0x64;         // 十六进制，以0x开头
        int binary = 0b1100100; // 二进制，以0b开头（Java 7+）
        
        System.out.println(decimal);
        System.out.println(octal);
        System.out.println(hex);
        System.out.println(binary);
    }
}

整数溢出

当整数超出其类型的范围时，会发生溢出：

public class Main {
    public static void main(String[] args) {
        int max = Integer.MAX_VALUE;
        System.out.println("最大值: " + max);
        System.out.println("溢出后: " + (max + 1));  // 会变成负数
    }
}

浮点类型

Java提供了两种浮点类型来表示小数：

类型 大小 精度 默认值
float 4字节 约7位有效数字 0.0f
double 8字节 约15位有效数字 0.0

浮点数采用IEEE 754标准存储，包含符号位、指数位和尾数位。

public class Main {
    public static void main(String[] args) {
        float f = 3.14f;        // float类型需要在数字后面加f
        double d = 3.14159265;
        
        System.out.println("float: " + f);
        System.out.println("double: " + d);
        
        // 科学计数法
        double scientific = 1.23e-4;  // 等于0.000123
        System.out.println("科学计数法: " + scientific);
    }
}

浮点数的精度问题

由于浮点数的存储方式，可能会出现精度问题：

public class Main {
    public static void main(String[] args) {
        double d1 = 0.1;
        double d2 = 0.2;
        double sum = d1 + d2;
        
        System.out.println(sum);  // 可能不是0.3
        System.out.println(sum == 0.3);  // false
    }
}

字符类型

Java使用char类型来表示单个字符：

public class Main {
    public static void main(String[] args) {
        char c1 = 'A';          // 字符常量用单引号
        char c2 = '中';
        char c3 = 65;           // ASCII码值
        char c4 = '\u0041';     // Unicode编码
        
        System.out.println(c1);
        System.out.println(c2);
        System.out.println(c3);
        System.out.println(c4);
    }
}

字符编码

计算机中字符的编码经历了多个发展阶段：

  • ASCII：美国标准信息交换码，7位编码，128个字符
  • GB2312：中国国家标准，支持简体中文
  • GBK：扩展的GB2312，支持更多中文字符
  • Unicode：国际标准，支持世界上所有语言
  • UTF-8：Unicode的变长编码实现
  • UTF-16：Unicode的定长编码实现

Java内部使用UTF-16编码存储字符，每个char占用2个字节。

布尔类型

布尔类型boolean只有两个值：true和false：

public class Main {
    public static void main(String[] args) {
        boolean flag1 = true;
        boolean flag2 = false;
        boolean result = 5 > 3;  // 比较运算的结果
        
        System.out.println(flag1);
        System.out.println(flag2);
        System.out.println(result);
    }
}

布尔类型主要用于条件判断和逻辑运算。
