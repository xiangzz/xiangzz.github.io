对象的使用

对象的引用

在Java中，对象变量实际上存储的是对象的引用（地址），而不是对象本身：

public static void main(String[] args) {
    Person person1 = new Person();
    Person person2 = person1;   // person2和person1指向同一个对象
    
    person1.name = "小明";
    System.out.println(person2.name);   // 输出：小明
}

对象变量存储的是对象的引用

点操作符

我们使用点操作符（.）来访问对象的属性和方法：

Person person = new Person();
person.name = "小红";        // 访问属性
person.age = 20;
person.hello();             // 调用方法

null值

如果对象变量没有指向任何对象，它的值就是null：

Person person = null;   // person不指向任何对象
if (person != null) {
    person.hello();     // 避免空指针异常
}

null表示不指向任何对象

属性的初始值

当我们创建对象时，如果没有给属性赋值，它们会有默认的初始值：

public class Student {
    String name;     // 默认值：null
    int age;         // 默认值：0
    boolean isGood;  // 默认值：false
    
    public void showInfo() {
        System.out.println("姓名：" + name);      // null
        System.out.println("年龄：" + age);       // 0
        System.out.println("是否优秀：" + isGood); // false
    }
}

不同类型属性的默认值
