3. 方法

3.1 方法的定义

方法是类中定义的函数，用于实现特定的功能。方法可以接收参数，执行操作，并可能返回结果。

public class Calculator {
    // 无返回值的方法
    public void printWelcome() {
        System.out.println("欢迎使用计算器！");
    }
    
    // 有返回值的方法
    public int add(int a, int b) {
        return a + b;
    }
    
    // 带多个参数的方法
    public double calculateArea(double length, double width) {
        return length * width;
    }
}

方法定义的语法结构

3.2 方法的调用

方法必须通过对象来调用（静态方法除外）。调用方法时需要提供相应的参数。

public class TestCalculator {
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        
        // 调用无返回值的方法
        calc.printWelcome();
        
        // 调用有返回值的方法
        int result = calc.add(5, 3);
        System.out.println("5 + 3 = " + result);
        
        // 调用带多个参数的方法
        double area = calc.calculateArea(4.5, 3.2);
        System.out.println("面积 = " + area);
    }
}

3.3 参数传递

Java中的参数传递都是按值传递的。对于基本数据类型，传递的是值的副本；对于引用类型，传递的是引用的副本。

public class ParameterTest {
    // 基本类型参数传递
    public void changeValue(int x) {
        x = 100;  // 不会影响原始值
        System.out.println("方法内 x = " + x);
    }
    
    // 引用类型参数传递
    public void changeArray(int[] arr) {
        arr[0] = 999;  // 会影响原始数组
        System.out.println("方法内 arr[0] = " + arr[0]);
    }
    
    public static void main(String[] args) {
        ParameterTest test = new ParameterTest();
        
        int num = 10;
        test.changeValue(num);
        System.out.println("方法外 num = " + num);  // 仍然是10
        
        int[] array = {1, 2, 3};
        test.changeArray(array);
        System.out.println("方法外 array[0] = " + array[0]);  // 变成999
    }
}

Java参数传递机制

3.4 this关键字

this关键字代表当前对象的引用，主要用于区分成员变量和局部变量，或者在方法中调用其他方法。

public class Student {
    private String name;
    private int age;
    
    // 使用this区分成员变量和参数
    public void setName(String name) {
        this.name = name;  // this.name是成员变量，name是参数
    }
    
    public void setAge(int age) {
        this.age = age;
    }
    
    // 使用this调用其他方法
    public void setInfo(String name, int age) {
        this.setName(name);
        this.setAge(age);
    }
    
    public void printInfo() {
        System.out.println("姓名：" + this.name + "，年龄：" + this.age);
    }
}

3.5 方法重载

方法重载是指在同一个类中定义多个同名方法，但参数列表不同（参数个数、类型或顺序不同）。

public class MathUtils {
    // 两个整数相加
    public int add(int a, int b) {
        return a + b;
    }
    
    // 三个整数相加
    public int add(int a, int b, int c) {
        return a + b + c;
    }
    
    // 两个浮点数相加
    public double add(double a, double b) {
        return a + b;
    }
    
    // 字符串拼接
    public String add(String a, String b) {
        return a + b;
    }
    
    public static void main(String[] args) {
        MathUtils utils = new MathUtils();
        
        System.out.println(utils.add(1, 2));           // 调用第一个方法
        System.out.println(utils.add(1, 2, 3));        // 调用第二个方法
        System.out.println(utils.add(1.5, 2.5));       // 调用第三个方法
        System.out.println(utils.add("Hello", "World")); // 调用第四个方法
    }
}

3.6 递归调用

递归是指方法调用自身的编程技巧。递归必须有终止条件，否则会导致栈溢出。

public class RecursionExample {
    // 计算阶乘
    public long factorial(int n) {
        if (n <= 1) {
            return 1;  // 终止条件
        }
        return n * factorial(n - 1);  // 递归调用
    }
    
    // 计算斐波那契数列
    public int fibonacci(int n) {
        if (n <= 1) {
            return n;  // 终止条件
        }
        return fibonacci(n - 1) + fibonacci(n - 2);  // 递归调用
    }
    
    public static void main(String[] args) {
        RecursionExample example = new RecursionExample();
        
        System.out.println("5! = " + example.factorial(5));
        System.out.println("第10个斐波那契数 = " + example.fibonacci(10));
    }
}
