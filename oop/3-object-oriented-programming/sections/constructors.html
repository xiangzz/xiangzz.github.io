4. 构造方法

4.1 构造方法的概念

构造方法是一种特殊的方法，用于创建对象时初始化对象的状态。构造方法的名称必须与类名相同，且没有返回类型。

public class Person {
    private String name;
    private int age;
    
    // 无参构造方法
    public Person() {
        this.name = "未知";
        this.age = 0;
        System.out.println("调用无参构造方法");
    }
    
    // 带参构造方法
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
        System.out.println("调用带参构造方法");
    }
    
    public void displayInfo() {
        System.out.println("姓名：" + name + "，年龄：" + age);
    }
}

构造方法的基本概念

4.2 无参构造方法

如果类中没有定义任何构造方法，Java会自动提供一个默认的无参构造方法。但如果定义了其他构造方法，就必须显式定义无参构造方法。

public class TestConstructor {
    public static void main(String[] args) {
        // 使用无参构造方法创建对象
        Person person1 = new Person();
        person1.displayInfo();
        
        // 使用带参构造方法创建对象
        Person person2 = new Person("张三", 25);
        person2.displayInfo();
    }
}

默认构造方法的工作原理

4.3 带参构造方法

带参构造方法允许在创建对象时传入参数来初始化对象的属性，提供了更灵活的对象创建方式。

public class Car {
    private String brand;
    private String model;
    private int year;
    private double price;
    
    // 带参构造方法1
    public Car(String brand, String model) {
        this.brand = brand;
        this.model = model;
        this.year = 2023;
        this.price = 0.0;
    }
    
    // 带参构造方法2
    public Car(String brand, String model, int year, double price) {
        this.brand = brand;
        this.model = model;
        this.year = year;
        this.price = price;
    }
    
    public void displayCarInfo() {
        System.out.println(brand + " " + model + " (" + year + ") - ￥" + price);
    }
}

带参构造方法的使用

4.4 构造方法重载

一个类可以有多个构造方法，只要它们的参数列表不同，这称为构造方法重载。

public class Rectangle {
    private double width;
    private double height;
    
    // 无参构造方法 - 创建单位正方形
    public Rectangle() {
        this.width = 1.0;
        this.height = 1.0;
    }
    
    // 一个参数构造方法 - 创建正方形
    public Rectangle(double side) {
        this.width = side;
        this.height = side;
    }
    
    // 两个参数构造方法 - 创建矩形
    public Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }
    
    public double getArea() {
        return width * height;
    }
    
    public static void main(String[] args) {
        Rectangle rect1 = new Rectangle();           // 1x1正方形
        Rectangle rect2 = new Rectangle(5);          // 5x5正方形
        Rectangle rect3 = new Rectangle(3, 4);       // 3x4矩形
        
        System.out.println("面积1：" + rect1.getArea());
        System.out.println("面积2：" + rect2.getArea());
        System.out.println("面积3：" + rect3.getArea());
    }
}

4.5 成员变量的初始化

成员变量可以在声明时直接初始化，也可以在构造方法中初始化。初始化的顺序是：声明时的初始化 → 构造方法中的初始化。

public class InitializationExample {
    // 声明时初始化
    private String name = "默认姓名";
    private int count = 0;
    private static int totalCount = 0;
    
    // 实例初始化块
    {
        count++;
        System.out.println("实例初始化块执行，count = " + count);
    }
    
    // 静态初始化块
    static {
        totalCount = 100;
        System.out.println("静态初始化块执行，totalCount = " + totalCount);
    }
    
    public InitializationExample() {
        this.name = "构造方法设置的姓名";
        System.out.println("构造方法执行，name = " + name);
    }
    
    public InitializationExample(String name) {
        this.name = name;
        System.out.println("带参构造方法执行，name = " + name);
    }
    
    public static void main(String[] args) {
        System.out.println("创建第一个对象：");
        InitializationExample obj1 = new InitializationExample();
        
        System.out.println("\n创建第二个对象：");
        InitializationExample obj2 = new InitializationExample("自定义姓名");
    }
}

4.6 this()调用其他构造方法

在一个构造方法中可以使用this()来调用同一个类的其他构造方法，但this()必须是构造方法的第一条语句。

public class Student {
    private String name;
    private int age;
    private String major;
    private String school;
    
    // 最完整的构造方法
    public Student(String name, int age, String major, String school) {
        this.name = name;
        this.age = age;
        this.major = major;
        this.school = school;
        System.out.println("调用完整构造方法");
    }
    
    // 调用完整构造方法，设置默认学校
    public Student(String name, int age, String major) {
        this(name, age, major, "默认大学");
        System.out.println("调用三参数构造方法");
    }
    
    // 调用三参数构造方法，设置默认专业
    public Student(String name, int age) {
        this(name, age, "计算机科学");
        System.out.println("调用二参数构造方法");
    }
    
    // 调用二参数构造方法，设置默认年龄
    public Student(String name) {
        this(name, 18);
        System.out.println("调用一参数构造方法");
    }
    
    public void displayInfo() {
        System.out.println("学生信息：" + name + ", " + age + "岁, " + major + ", " + school);
    }
    
    public static void main(String[] args) {
        Student student = new Student("李四");
        student.displayInfo();
    }
}
