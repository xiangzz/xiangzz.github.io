什么是 Lambda 表达式？

Lambda 表达式是 Java 8 引入的一个重要特性，它允许我们以更简洁的方式编写匿名函数。

传统写法 😰

// 对字符串列表进行排序
List<String> names = Arrays.asList("张三", "李四", "王五");
Collections.sort(names, new Comparator<String>() {
    @Override
    public int compare(String a, String b) {
        return a.compareTo(b);
    }
});

Lambda 写法 😍

// 同样的功能，更简洁
List<String> names = Arrays.asList("张三", "李四", "王五");
Collections.sort(names, (a, b) -> a.compareTo(b));

Lambda 表达式让代码更简洁、更易读、更函数式！

Lambda 表达式语法

基本语法结构

(参数列表) -> { 方法体 }
参数列表 可以为空，单个参数可省略括号
箭头操作符 分隔参数和方法体
方法体 单行可省略大括号和return

语法示例

// 无参数
() -> System.out.println("Hello")

// 单个参数（可省略括号）
x -> x * 2
(x) -> x * 2

// 多个参数
(x, y) -> x + y

// 多行方法体
(x, y) -> {
    int sum = x + y;
    return sum * 2;
}

函数式接口

Lambda 表达式只能用于函数式接口（只有一个抽象方法的接口）

自定义函数式接口

// 定义一个计算接口
interface Calculator {
    int calculate(int a, int b);
}

// 传统方式
Calculator add = new Calculator() {
    public int calculate(int a, int b) {
        return a + b;
    }
};

// Lambda 方式
Calculator add = (a, b) -> a + b;

Comparator

// 传统方式
Collections.sort(list, new Comparator<String>() {
    public int compare(String a, String b) {
        return a.compareTo(b);
    }
});

// Lambda 方式
Collections.sort(list, (a, b) -> a.compareTo(b));

常用函数式接口

1. Predicate<T>

功能：接受一个参数，返回 boolean 值（用于判断条件）

// 定义判断条件
Predicate<String> isEmpty = s -> s.isEmpty();
Predicate<Integer> isEven = n -> n % 2 == 0;
Predicate<Integer> isPositive = n -> n > 0;

// 使用示例
System.out.println(isEmpty.test(""));        // true
System.out.println(isEven.test(4));          // true
System.out.println(isPositive.test(-5));     // false

2. Function<T, R>

功能：接受一个参数，返回一个结果（用于数据转换）

// 定义转换函数
Function<String, Integer> getLength = s -> s.length();
Function<Integer, String> toString = n -> String.valueOf(n);
Function<String, String> toUpper = s -> s.toUpperCase();

// 使用示例
System.out.println(getLength.apply("Hello"));    // 5
System.out.println(toString.apply(123));         // "123"
System.out.println(toUpper.apply("world"));      // "WORLD"

3. Consumer<T>

功能：接受一个参数，无返回值（用于执行操作）

// 定义消费操作
Consumer<String> print = s -> System.out.println(s);
Consumer<Integer> doubleAndPrint = n -> System.out.println(n * 2);
Consumer<String> addPrefix = s -> System.out.println("前缀: " + s);

// 使用示例
print.accept("Hello World");           // 输出: Hello World
doubleAndPrint.accept(5);              // 输出: 10
addPrefix.accept("测试");               // 输出: 前缀: 测试

4. Supplier<T>

功能：无参数，返回一个结果（用于提供数据）

// 定义供应商
Supplier<String> greeting = () -> "Hello World";
Supplier<Integer> randomNum = () -> (int)(Math.random() * 100);
Supplier<String> currentTime = () -> new Date().toString();

// 使用示例
System.out.println(greeting.get());      // Hello World
System.out.println(randomNum.get());     // 随机数 0-99
System.out.println(currentTime.get());   // 当前时间

Lambda 表达式的基本应用

Lambda 表达式让集合操作变得更简洁、更优雅

简单应用示例

List<String> names = Arrays.asList("张三", "李四", "王五", "赵六");

// 传统方式：遍历并打印每个名字
for (String name : names) {
    System.out.println("姓名: " + name);
}

// Lambda 方式：更简洁的遍历
names.forEach(name -> System.out.println("姓名: " + name));

// 或者使用方法引用（更简洁）
names.forEach(System.out::println);

Lambda 的基本应用

  • forEach() - 遍历集合中的每个元素
  • sort() - 对集合进行排序
  • 自定义接口 - 创建自己的函数式接口
  • 方法引用 - 简化 Lambda 表达式

方法引用

当 Lambda 表达式只是调用一个现有方法时，可以使用方法引用让代码更简洁

静态方法引用

// Lambda 表达式
Function<String, Integer> parser = s -> Integer.parseInt(s);

// 方法引用
Function<String, Integer> parser = Integer::parseInt;

实例方法引用

// Lambda 表达式
Consumer<String> printer = s -> System.out.println(s);

// 方法引用
Consumer<String> printer = System.out::println;

构造器引用

// Lambda 表达式
Supplier<List<String>> listSupplier = () -> new ArrayList<>();

// 构造器引用
Supplier<List<String>> listSupplier = ArrayList::new;

实战应用

数字计算器应用

// 定义计算器接口
interface Calculator {
    int calculate(int a, int b);
}

// 使用 Lambda 表达式创建不同的计算器
Calculator add = (a, b) -> a + b;
Calculator subtract = (a, b) -> a - b;
Calculator multiply = (a, b) -> a * b;
Calculator divide = (a, b) -> a / b;

// 使用计算器
int result1 = add.calculate(10, 5);        // 15
int result2 = subtract.calculate(10, 5);   // 5
int result3 = multiply.calculate(10, 5);   // 50
int result4 = divide.calculate(10, 5);     // 2

// 对数字列表进行排序
List<Integer> numbers = Arrays.asList(5, 2, 8, 1, 9);
Collections.sort(numbers, (a, b) -> a.compareTo(b)); // 升序
Collections.sort(numbers, (a, b) -> b.compareTo(a)); // 降序

Lambda 表达式最佳实践

✅ 保持简洁

// 好的做法
list.forEach(System.out::println);

// 避免过于复杂的 Lambda
list.forEach(item -> {
    // 很多行代码...
    // 考虑提取为单独的方法
});

✅ 使用方法引用

// 推荐使用方法引用
names.forEach(System.out::println);

// 而不是
names.forEach(name -> System.out.println(name));

✅ 选择合适的场景

// 适合使用 Lambda：简单的操作
Collections.sort(numbers, (a, b) -> a.compareTo(b));

// 复杂逻辑建议使用传统方法
// 保持代码的可读性和可维护性
